Keysight N6841A RF Sensors, Fortem TrueView R20 Radar, Keysight Geolocation Software

Description

This dataset contains measurements from UAV tracking experiments conducted across 33 AADM ( AERPAW Autonomous Data Mule) flights at the Lake Wheeler field site. These experiments were designed to evaluate multi-modal UAV detection and tracking performance using passive Keysight N6841A RF sensors and a Fortem TrueView R20 radar. In addition to the AADM missions, three supplementary flights were performed specifically to optimize RF sensor placement, geometry, and signal reception. The combined dataset provides ground-truth trajectories, RF sensor position estimates, and radar measurements for UAVs operating under diverse flight patterns.

Potential use cases for this dataset include:

  • Benchmarking RF-based UAV localization algorithms
  • Evaluating tracking and filtering methods
  • Developing multi-sensor fusion approaches
  • Validating theoretical localization bounds in real flight conditions
