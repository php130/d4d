## Notes For Hackathon Selection

- If time is short, skip custom OSDK and focus on CSV ingestion, ontology, Workshop, AIP Analyst/Agent, and report generation.
- If the team needs a polished command-center demo, prioritize Common Operating Picture, alerting workflow, embedded AIP Agent, and Vertex Graph examples.
- If the project is T2-heavy, prioritize RAG evals, semantic search, AIP Logic classification, document parsing, and graph exploration.
- If the project is T4-heavy, prioritize geospatial transformation, COP, maps, time series, and anomaly detection examples.
